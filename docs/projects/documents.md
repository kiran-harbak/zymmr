# Project Documents

::: info Overview
The Project Documents feature serves as a centralized repository for all files related to a specific project. Located within the **Project Settings**, it allows team members to upload specs, designs, contracts, and other essential resources, ensuring access to the latest documentation in one place.
:::

## 1. Accessing Project Documents

1.  Navigate to your specific **Project**.
2.  Click on the external link icon <svg xmlns="http://www.w3.org/2000/svg" fill="none" viewBox="0 0 24 24" stroke-width="2.0" stroke="currentColor" class="navigation-icon"><path stroke-linecap="round" stroke-linejoin="round" d="M13.5 6H5.25A2.25 2.25 0 003 8.25v10.5A2.25 2.25 0 005.25 21h10.5A2.25 2.25 0 0018 18.75V10.5m-10.5 6L21 3m0 0h-5.25M21 3v5.25"></path></svg> located next to the project name to open **Project Settings**.
3.  In the left-hand sidebar, click on **Documents**.

## 2. The Documents Dashboard

This screen lists all files currently stored in the project.

-   **+ Upload Document Button**: Located in the top right, this button initiates the file upload process.

The dashboard table displays the following details:

| Column       | Description                                                                 |
| :----------- | :-------------------------------------------------------------------------- |
| **FILE NAME**| The name and format of the uploaded file (e.g., `specs_v2.pdf`, `logo.png`).|
| **SIZE**     | The file size (e.g., `7.3 MB`).                                             |
| **UPLOADED BY**| The team member who uploaded the file.                                    |
| **UPLOADED** | A timestamp indicating when the file was added.                             |
| **ACTIONS**  | Icons to **Download** (arrow) or **Delete** (trash can) the file.           |

## 3. Uploading and Managing Files

### Uploading a Document
1.  Click the **+ Upload Document** button.
2.  A file selection window will open. Browse your computer to find the file you wish to upload.
3.  Select the file and click **Open**.
4.  The file will be uploaded immediately and will appear in the documents list.

### Managing Existing Files
-   **Download**: To view or save a file to your computer, click the **Download** (down arrow) icon in the file's row.
-   **Delete**: To remove a file from the project, click the **Delete** (trash can) icon.

::: tip Best Practice
Encourage your team to use clear, descriptive file names before uploading. This makes it much easier for other members to find the correct document later (e.g., use `Project_Proposal_Final.pdf` instead of `doc1.pdf`).
:::
